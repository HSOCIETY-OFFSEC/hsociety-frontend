# Hsociety OffSec

Hsociety OffSec is a technology-driven platform focused on **offensive security**, **ethical hacking**, and **tech training**. The platform aims to provide hands-on learning experiences for aspiring cybersecurity enthusiasts and developers.

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/hsociety-offsec.git
cd hsociety-offsec
Install dependencies:

npm install


Start the development server:

npm run dev


Open http://localhost:5173
 to view in your browser.

Usage

Update the index.html in the public folder for SEO and favicon changes.

All React components are in the src folder.

Make sure to place the hsociety-logo-white.png favicon in the public folder.
